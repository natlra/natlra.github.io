2023 Screwdriver Project

Heat and Manufacturing

Project Overview

This page provides an overview of the Screwdriver Project, which focuses on manufacturing techniques involving heat.

About This Project

This project concentrated on manufacturing techniques that require the heating of materials to modify its shape, structural integrity, and appearance. These different techniques are important to manufacturing because they provide a way to change multiple different aspects of a material using the same approach. Being able to repeat a process for multiple stages of manufacturing makes some operations much more efficient and reduces costs related to manufacturing.

  • Labor Cost: $24.77 x ~5 hours = $123.85
  • Total Individual Screwdriver Cost: $145.15

Reducing Manufacturing Costs

Efficient manufacturing practices can help reduce costs. Consider the following strategies when manufacturing these screwdrivers at a large scale:

  • Assembly Line manufacturing: If each person is trained to do one step of the screwdriver, we can reduce the labor cost
  • Buy bulk raw stock to reduce individual material cost.
  • Try and match the demand curve with supply.
